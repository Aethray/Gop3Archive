### 2. 翻前动作决策矩阵
> 系统会根据当前大盲注级别和面临的下注额，动态判定。(免费看牌时永远选择“过牌”)

| 盲注级别 | 全押标准(All-in) | 小注(≤200) | 中注(≤400) | 大注(>400) |
| :--- | :--- | :--- | :--- | :--- |
| **≤ 50** | ≥ 16分 / 对7+ | ≥ 9分跟 | ≥ 11分跟 | ≥ 12分跟 |
| **≤ 100**| ≥ 16分 / 对7+ | ≥ 7分跟 | ≥ 9分跟 | ≥ 11分跟 |
| **≤ 200**| ≥ 13分 / 对5+ | ≥ 7分跟 | ≥ 8分跟 | ≥ 10分跟 |
| **> 200**| ≥ 11分 / 任一对 | ≥ 7分跟 | ≥ 8分跟 | ≥ 6分(博弈)|

## 🌪️ 二、翻牌圈策略 (Flop)
> **判定逻辑**: 此时放弃翻前分数，进入【牌面档位 (Tier)】物理级判定。

* **超强**: 双对、三条、顺子、同花以上。
* **强牌**: 最大顶对带听牌、大口袋对。
* **中等牌1/2**: 中对子、底对。
* **听牌1/2**: 两头顺、听同花、卡顺。

| 场面局势 | 超强牌 | 强牌/中等1/听牌2 | 中等2/听牌1 | 垃圾牌 |
| :--- | :--- | :--- | :--- | :--- |
| **白嫖 (没人下注)** | 最小加注 (价值) | 全押 (强势收池) | 过牌 | 过牌 |
| **盲注低，面临下注**| 最小加注 | 全押 (反推) | ≤200跟，贵则弃 | 弃牌 |
| **盲注中，面临下注**| 中等加注 | 全押 (反推) | ≤300跟，贵则弃 | 弃牌 |
| **盲注高，面临下注**| 全押 | 全押 | 高盲注下全押博弈 | 弃牌 |

## 📈 三、转牌圈策略 (Turn)
> **核心**: 公共牌发出4张。听牌成型概率减半，对“听牌”和“中对子”要求极其苛刻。新增档位：双对强、双对中、公牌高概率平局牌。

| 场面局势 | 超强/强牌 | 双对强 | 中等1/强听牌 | 弱对/弱听/垃圾 |
| :--- | :--- | :--- | :--- | :--- |
| **死局面(牌面一样大)**| 跟注 / 全押(无注)| 同左 | 同左 | 同左 |
| **白嫖 (没人下注)** | 全押 (梭哈) | 中等加注 | 最小加注 | 过牌 |
| **面临一般下注** | 全押 (遇注反推) | 中等加注 / 价值打| 跟注看最后一张 | 弃牌 |
| **面临高额下注** | 全押 (硬碰硬) | 全押 | 咬牙硬跟 | 弃牌止损 |

## 🏁 四、河牌圈策略 (River)
> **核心**: 公共牌发完(5张)。彻底关闭所有听牌逻辑（没中即垃圾），启动危险面降级机制（如桌面有顺花可能，单对自动降级）。

| 场面局势 | 超强牌 | 双对强 | 强/双对弱/中等 | 极弱对/垃圾牌 |
| :--- | :--- | :--- | :--- | :--- |
| **死局面(牌面一样大)**| 跟注平分 / 全押 | 同左 | 同左 | 同左 |
| **白嫖 (没人下注)** | 全押 (绝杀) | 中等加注 | 最小加注 (榨取) | 过牌 |
| **低盲注，面临下注**| 全押 | 全押 / 遇注反推| 便宜则跟注看牌 | 弃牌 |
| **高盲注，面临下注**| 全押 | 全押 | 咬牙硬跟 (底池大)| 弃牌止损 |

## ⚙️ 五、操作动作对应说明
引擎决策出动作后，物理点击操作的映射逻辑如下：

* 🚀 **全押 (All-in)**：直接点击全押按钮（或筹码不足时点击最高额度）。
* 💰 **加注2 (重加注)**：计算 `当前底池 + 需跟注额`。如果所占自身筹码比例不高，则下注 **满底池 (Pot)**；否则下注 **3/4 底池**。
* 💵 **加注1 (轻加注)**：下注 **1/2 底池**。
* 🤙 **跟注 (Call)**：点击跟注/平跟按钮。
* ✋ **过牌 (Check)**：点击过牌按钮。
* 🖱️ **弃牌 (Fold)**：点击弃牌按钮（弃牌后自动执行防封号的老虎机/互动逻辑）。
