### 2. 翻前动作决策矩阵
系统会根据当前**大盲注级别**和**面临的下注额**，动态判定这手牌是全押、跟注还是弃牌。
*(免费看牌时永远选择“过牌”)*

| 当前盲注级别 | 全押标准 (直接All-in) | 面临小下注 (≤200) | 面临中下注 (≤400) | 面临大下注 (>400) |
| :--- | :--- | :--- | :--- | :--- |
| **盲注 ≤ 50** | ≥ 16分 (或 对7以上) | ≥ 9分 才跟 | ≥ 11分 才跟 | ≥ 12分 才跟 |
| **盲注 ≤ 100** | ≥ 16分 (或 对7以上) | ≥ 7分 才跟 | ≥ 9分 才跟 | ≥ 11分 才跟 |
| **盲注 ≤ 200** | ≥ 13分 (或 对5以上) | ≥ 7分 才跟 | ≥ 8分 才跟 | ≥ 10分 才跟 |
| **盲注 > 200** | ≥ 11分 (或 任意对子) | ≥ 7分 才跟 | ≥ 8分 才跟 | ≥ 6分 (短码博弈跟注) |

## 二、 翻牌圈策略 (Flop)

公共牌发出 3 张后，放弃翻前分数，改用 **“牌面档位 (Tier)”** 进行物理级判定。

* **超强**：双对、三条、顺子、同花以上。
* **强牌**：最大顶对带听牌、大口袋对。
* **中等牌1/2**：中对子、底对。
* **听牌1/2**：两头顺、听同花、卡顺。

| 场面局势 | 你持有：超强 | 你持有：强牌 / 中等牌1 / 听牌2 | 你持有：中等牌2 / 听牌1 | 你持有：垃圾牌 |
| :--- | :--- | :--- | :--- | :--- |
| **白嫖 (没人下注)** | 最小加注 (打价值) | 全押 (强势收下底池) | 过牌 | 过牌 |
| **盲注低，面临下注** | 最小加注 | 全押 (反推) | 便宜跟注(≤200)，贵了弃牌 | 弃牌 |
| **盲注中，面临下注** | 中等加注 | 全押 (反推) | 便宜跟注(≤300)，贵了弃牌 | 弃牌 |
| **盲注高，面临下注** | 全押 | 全押 | 高盲注下全押博弈 | 弃牌 |

## 三、 转牌圈策略 (Turn)

公共牌发出 4 张。此时听牌成型的概率减半，策略对“听牌”和“中等对子”的要求变得更加苛刻。

* **新增档位**：双对强、双对中、公牌高概率平局牌 (如桌面发出皇家同花顺)。

| 场面局势 | 你持有：超强 / 强牌 | 你持有：双对强 | 你持有：中等牌1 / 强听牌 | 弱对子/弱听/垃圾 |
| :--- | :--- | :--- | :--- | :--- |
| **死局面 (牌面一样大)**| 跟注(有人下注) / 全押(没人下注白捡) | 同左 | 同左 | 同左 |
| **白嫖 (没人下注)** | 全押 (梭哈拿钱) | 中等加注 | 最小加注 | 过牌 |
| **面临一般下注** | 全押 (遇注反推) | 中等加注 / 价值打 | 跟注看最后一张 | 弃牌 |
| **面临高额下注** | 全押 (高注硬碰硬) | 全押 | 咬牙硬跟 | 弃牌止损 |

## 四、 河牌圈策略 (River)

公共牌全部发完（5张）。此时**彻底关闭所有听牌逻辑**（没中就是垃圾），并启动**危险面降级机制**（如果桌面有同花或顺子的可能，普通的顶对会被降级处理）。

* **档位简化为**：超强、双对强、双对弱、强、中等1/2/3、垃圾。

| 场面局势 | 你持有：超强 | 你持有：双对强 | 你持有：强 / 双对弱 / 中等 | 极弱对子 / 垃圾牌 |
| :--- | :--- | :--- | :--- | :--- |
| **死局面 (牌面一样大)**| 跟注平分 / 没人下注则全押 | 同左 | 同左 | 同左 |
| **白嫖 (没人下注)** | 全押 (绝杀) | 中等加注 | 最小加注 (榨取剩余价值) | 过牌 |
| **盲注低，面临下注**| 全押 | 全押 / 遇注反推 | 便宜则跟注看牌 | 弃牌 |
| **盲注高，面临下注**| 全押 | 全押 | 咬牙硬跟 (因为底池通常已经很大)| 弃牌止损 |

## 五、 操作动作对应说明

引擎决策出动作后，物理点击操作的映射逻辑如下：
* **全押 (All-in)**：直接点击全押按钮（或筹码不足时点击最高额度）。
* **加注2 (重加注)**：计算 `当前底池 + 需跟注额`。如果所占自身筹码比例不高，则下注 **满底池 (Pot)**；否则下注 **3/4 底池**。
* **加注1 (轻加注)**：下注 **1/2 底池**。
* **跟注 (Call)**：点击跟注/平跟按钮。
* **过牌 (Check)**：点击过牌按钮。
* **弃牌 (Fold)**：点击弃牌按钮（弃牌后自动执行防封号的老虎机/互动逻辑）。
